Introduction

This is the example page for sidebar-links. It will take <h1> to <h6> tags with a specified class (like the title of this section) and create <li> items (and nested <ul> items) in the specified <ul> element. The sidebar on the right was generated sidebar-links.js.

Excluding Headers

If a header should not be included in the list, don't add the specified class name to it. Notice the header for this section is not included in the sidebar.

Example

            
<script type="text/javascript">
    document.addEventListener("DOMContentLoaded", function() {
        // Call sidebarLinks with:
        //   * the ul id name
        //   * the header element class name
        //   * the max nesting depth. Since there are only <h1> to <h6> tags, this can't be more than 6.
        // The values called below are the defaults.
        sidebarLinks('sidebar-links', 'sidebar-link', 6);
    });
</script>
